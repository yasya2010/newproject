# 🎬 Movie Library - Личная кинотека

**Автор:** Дмитрий Шагунов

## Описание
Это приложение для хранения любимых фильмов. Добавляй фильмы, 
ставь рейтинг, фильтруй по жанру или году. Данные сохраняются 
в JSON-файл, так что после перезапуска ничего не пропадает.

## Как пользоваться
1. Заполни поля: название, жанр, год (только число), рейтинг (0-10)
2. Нажми "Добавить фильм" — он появится в таблице
3. Чтобы удалить — выбери строку и нажми "Удалить"
4. Фильтруй: выбери жанр или введи год в поле сверху

## Примеры тестов
✅ Добавить "Начало", жанр "Фантастика", 2010, рейтинг 9.5  
✅ Добавить "Криминальное чтиво", жанр "Криминал", 1994, рейтинг 9.0  
✅ Фильтр по жанру "Фантастика" → покажет только "Начало"  
✅ Фильтр по году 1994 → покажет "Криминальное чтиво"  
❌ Если год не число → ошибка  
❌ Если рейтинг 11 → ошибка
